The Project:

Creating an implementation of a Bayesian network for recess detection using the sensor model of an iRobot Create 3. Then determine the robot's actual distance from the wall as well as when it is 10 cm past a door.

Baysian Network

We used the python library Pomegranate to build and fit our Bayesian network from gathered data. The data we gathered was of the robot heading toward the door with the wall on its left. The network at school proved a challenge when connecting to the robot via ROS. We ended up having to take the robot home and improvise a door to use for training.

Links

  • Measurements

    If you want to take measurements using the code you have to download all the workspaces and then source the setup files. As well as have ROS2 downloaded.

    The folder structure is as follows:

    • create3_coverage

      This is a workspace for the original example code provided by the Create team that we adapted into our data-gathering code.

    • create3_examples_ws

      This is a workspace for our data collection code adapted from the Create examples provided by the Create team.

    • create3_workspace

      This is a workspace for the Create robot simulator. We used this a few times when developing the code

    • door_data

      This is all of our collected data from the robot that we gathered during training.

    • ros2_ws

      This is a workspace that has the specialized irobot_create_msgs which are customized actions, topics, and messages that the robot uses to communicate in ROS.

    • ros_ign

      This is a workspace that contains code for linking ROS to Gazebo. This is used by the simulator code.

    • workspace

      This is a workspace containing Gazebo support code. This is used by the simulator code.
